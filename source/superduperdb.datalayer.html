class superduperdb.datalayer.query_dataset.CachedQueryDataset(select: Select, keys=None, fold='train', suppress=(), transform=None, features=None, database=None, prefetch_size: int = 100)[source]

Bases: Dataset

A cached dataset class which can be used to define a torch dataset class. This class which fetch the document corresponding to the given index. This class prefetches documents from database and stores in the memory.

This can drastically reduce database read operations and hence reduce the overall load on the database.
